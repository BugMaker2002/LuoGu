# 位运算
## 求n的二进制表示中第k位是几：
### 1.先把第k位右移到最后一位：n>>k
### 2.看个位是几：与1相与&
### 不必将n化为二进制表示
## 代码模板：
```cpp
int n=12;
for (int k = 3; k >=0; k--) 
{
    cout<<(n>>k&1);
}
```
# lowbit(x)函数
## 作用：返回x当中的最后一位1
### 原理：x&-x，因为x的负数的二进制表示是x的反码加一
### 如x=1010，则lowbit(x)=10
### 如x=101000，则lowbit(x)=1000
## 求一个数的补码：
```cpp
int n=12;
unsigned int x = -n;
for (int k = 3; k >=0; k--) 
{
    cout<<(x>>k&1);
}
```
